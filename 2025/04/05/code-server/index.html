在我的安卓手机上安装Code(code-server)

安装termux

设置sshd

打开termux,安装sshd

运行sshd,sshd默认端口8022

获得手机ip,用户名，设置密码

从电脑上登陆手机，注意不要让手机锁屏

安装code-server

https://coder.com/docs/code-server/termux

运行code-server

不加参数直接运行code-server之绑定127.0.0.1:8080

浏览器访问http://<手机IP>:8080

页面提示

在config.yaml文件的最后可以找到密码
